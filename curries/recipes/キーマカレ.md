# 🍛 キーマカレー

## 材料
| 具材 | 量 |
| --- | --- |
| 液体酸素 | 100ml |
| レトルトカレーのルー | 1袋 |
| 玉ねぎ | 1個 |
| 人参 | 1本 |
| 鶏ひき肉 | 200g |

## 作り方
1. 玉ねぎと人参をみじん切りにします  
2. フライパンで鶏ひき肉を中火で炒めます  
3. 玉ねぎと人参を加え、しんなりするまで炒めます  
4. レトルトカレーのルーを加えて全体を混ぜます  
5. 粗熱を取り、耐冷容器に移します  
6. キアコンを用いて、液体酸素を少量ずつ加えながら急速冷却します  
7. 全体がキンキンに冷えたら器に盛り付けます  
8. 完成
